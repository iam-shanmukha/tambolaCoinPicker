# tambolaCoinPicker
Tambola Coin Picker written in Python using kivy </br>
# Usage
`$ git clone https://github.com/iam-shanmukha/tambolaCoinPicker.git`</br>
`$ cd tambolaCoinPicker`</br>
`$ pip install -r requirements.txt`</br>
`$ python3 main.py`</br>
# Download
Downlad Apk from /bin folder
